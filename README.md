# 🔗 URL Shortener

A full-stack URL shortener web application built using Node.js, Express, MongoDB, and EJS.

## 🚀 Features
- Shorten long URLs
- Custom short codes
- QR code generation
- Click tracking
- Delete URLs

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Bootstrap + Custom CSS
